1.Project Overview
This project focuses on Exploratory Data Analysis (EDA) of the Superstore sales dataset.
The objective is to understand the data, analyze sales and profit patterns, and identify meaningful insights before performing advanced analysis or modeling.

2. Dataset Description
The Superstore dataset contains retail sales transaction data.
It includes information related to products, sales, profit, and regions.
Key Columns in the Dataset:
Order ID – Unique order identifier
Order Date – Date of purchase
Category – Product category
Sub-Category – Detailed product classification
Region – Sales region
Sales – Revenue generated
Profit – Profit or loss per order
Quantity – Number of units sold

3.Tools Used
Python
Pandas
Matplotlib
Jupyter Notebook

4.EDA Steps Performed
Loaded and inspected the dataset
Checked data types and missing values
Analyzed sales and profit distribution
Used grouping and aggregation to identify:
Most profitable categories
Least profitable categories
Created basic visualizations to support analysis
